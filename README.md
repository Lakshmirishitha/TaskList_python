# To-Do List Application (Python CLI)

This is a simple console-based to-do list manager created for the Python Developer Internship Task 2.

## Features
- Add tasks
- View tasks
- Remove tasks
- Persistent storage using tasks.txt
- Simple CLI menu

## How to Run
1. Install Python
2. Open terminal inside the folder
3. Run:

   python todo.py

A tasks.txt file will be created automatically.
